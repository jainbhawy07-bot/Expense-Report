# Personal Expense Report

## Overview
A personal expense reporting and analysis system created using Microsoft Excel and Google Sheets.

## Features
- Daily expense recording
- Category-wise expense tracking
- Total expense calculation
- Average expense analysis
- Highest expense identification
- Expense visualization using charts

## Skills Used
- Microsoft Excel
- Google Sheets
- Data Analysis
- Data Entry
- Formulas
- Data Visualization
